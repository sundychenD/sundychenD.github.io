Project Description


Worked as a software engineer, I was assigned to design and build the front-end of enterprise client payload monitoring website.

The site tracks daily usage of Autodesk license checked out by enterprise clients. A line chart is displayed on top to give an usage overview. At the bottom, errors would be highlighted.

The site is built for business analysis team as well as the client support team for continuous monitoring of client license activity.
